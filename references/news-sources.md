## Source Selection Strategy

### For Daily Briefing (Standard Run)

1. **Always check** all Tier 1 sources (both international and domestic)
2. **Add 2-3** Tier 2 sources for completeness
3. **Use web search** to catch breaking news not yet on major sites

### For Focus Topics (Codex / Claude Code / Gemini)

1. **Always check** OpenAI Blog, Anthropic News, Google AI Blog first
2. Search specifically for these product names
3. Include any update, even minor ones

### Source URL Reminder

> **CRITICAL**: When extracting news from these sources, always get the
> **specific article URL** (e.g., `https://36kr.com/p/123456`),
> NOT the source homepage or category page listed above.
> The URLs above are for finding articles, not for linking in the output.
